■VC++でのビルド

VC++でビルドするためのライブラリ・ツールは以下から入手できます。

FFTW2.1.3
	インクルードファイル
		ftp://ftp.fftw.org/pub/fftw
	インポートライブラリ
		http://claymore.engineer.gvsu.edu/~steriana/software_fftw.html

Boost
	インクルードファイル (インクルードファイルのみで可)
		http://www.boost.org/

Qt3.2.1以降
	インクルードファイル・インポートライブラリ
	および各種開発用ツール群
		"C++ Gui Programming With Qt 3"  Prentice Hall Ptr  ISBN: 0131240722
		付属のCD-ROMからインストール
		または、http://kde-cygwin.sourceforge.net/qt3-win32/ から
		GPL版のWindow用Qt開発環境(ソース/バイナリ)を入手できる
